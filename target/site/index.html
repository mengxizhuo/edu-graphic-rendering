图形渲染系统

概述

图形渲染系统是一个基于 Java 的应用程序，用于演示在图形渲染环境中各种设计模式的实现。该系统允许用户创建、操作和渲染不同类型的图形，并支持多种渲染技术。

功能特点

  • 创建多种图形（圆形、矩形、线条、三角形）
  • 图形操作（移动、调整大小）
  • 使用不同的渲染器渲染图形（控制台、SVG、Swing）
  • 撤销/重做功能
  • 将图形导出为不同格式（XML、JSON）
